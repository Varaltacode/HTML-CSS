Principais formatações

Negrito

Podemos usar a tag b para ficar em negrito

Para ficar em negrito e semântico, use strong

Itálico

Para deixar em itálico, use i

Para deixar em itálico com semântica use em

Marca-texto

Para criar um texto marcado, use a tag mark

Assim

Também é possível editar a cor do marca-texto

Texto grande e pequeno

Para deixar um texto grande, use a tag big

Para deixar um texto pequeno, use a tag small

Texto deletado

Para marcar um texto que deve ser lido mas não considerado, use del

Texto incluido

Para marcar um texto sublinhado que foi inserido posteriormente, use ins

Também é possível usar o sublinhado com a tag U

Texto sobrescrito

Usado para inserir coisas do tipo x4+3 use a tag sup

Texto subscrito

Usado para inserir coisas do tipo H20 use a tag sub
